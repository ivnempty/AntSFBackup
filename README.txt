Steps for New Project Source Backup

1. Create project folder
2. Copy these files
3. Update build.properties (username, password, target, etc)
4. Create git repo in git server
5. Clone git repo to project folder
6. Run cmd as Administrator and switch to project folder (otherwise cannot push to Git server)
7. Run commit.bat to start backup, or
8. Setup schedule task and run it to start backup (refer to ScheduleTask_Setup_Sample.docx)



Refer to https://developer.salesforce.com/docs/atlas.en-us.daas.meta/daas/forcemigrationtool_install.htm
